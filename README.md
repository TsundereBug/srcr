# srcr

A Crystal wrapper for the Speedrun.com API.

## Installation

Add this to your `shard.yml`:

```yaml
dependencies:
  srcr:
    github: TsundereBug/srcr
```

## Usage

Everything can be accessed through the `SRcr` module. For example, `SRcr::Runs.forCategory(c)`
